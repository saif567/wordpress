== Description ==
Our 2015 default theme is clean, blog-focused, and designed for clarity. airline's simple, straightforward typography is readable on a wide variety of screen sizes, and suitable for multiple languages. We designed it using a mobile-first approach, meaning your content takes center-stage, regardless of whether your visitors arrive by smartphone, tablet, laptop, or desktop computer.

* Mobile-first, Responsive Layout
* Custom Colors
* Custom Header
* Social Links
* Menu Description
* Post Formats
* The GPL v2.0 or later license. :) Use it to make something cool.



== Installation ==

1. Download all these files and place it in your theme's folder of worpress within a new folder.
2. Now go to your admin panel or worpdress dashboard click on Appearance->Themes  thene select the theme and activate it.
